📌 About
This repository serves as a practical guide for learning the fundamentals of tsunami waveform inversion. 
This project breaks down the process of reconstructing fault slip distributions from recorded (or synthetic) sea-level data by utilizing synthetic bathymetry and COMCOT for forward-modeling the tsunami simulation.

🛠️ Directories
1. Batimetri/ : contains synthetic bathymetry data necessary for running tsunami propagation simulations.
2. GF/ : contains pre-computed tsunami waveforms generated by triggering a unit slip at specific sub-faults across a 2x2 grid matrix ($A1$, $A2$, $B1$, $B2$).
3. comcot_base/ : contains baseline configuration files for COMCOT for streamlining the generation of  Green's Function simulation runs.
4. obs_data : contains Contains the "true" or synthetic observation gauge data generated using the initial fault parameter targets found in source.txt

Python Scripts
1. comcot_to_csv.py: Converts raw COMCOT sea-level output into standard time-amplitude formats, downsampling/resampling the data to a uniform 1-minute sampling interval.
2. pick_arrival.py: Uses a threshold-based algorithm to automatically flag the first arrival of the tsunami wave at various stations. Outputs findings to sta_param.txt.
3. plot_waveform.py: A plotting utility to visualize and compare simulated Green's Functions against observation data.
4. create_matrix.py: Compiles processed waveform data into matrix structure. Saves the system matrix as Matrix_A_GF.csv and the observation vector as Matrix_B_OBS.csv.
5. inverse_matrix.py: Executes matrix inversion to solve for the unknown slip distribution multiplier vector ($x$). Outputs the final estimated slip for each unit_sources to Matrix_X_result.csv.
